Analyseur - Interpreteur en C++

L'exécutable Linux se trouve à la racine du projet. Pour le lancer, taper ./LTN en ligne de commande.

Pour construire l'exécutable, lancer la commande : make 
La cible est crée dans la répertoire /bin, les .o dans /build

Pour nettoyer le repertoire, lancer la commande : make clean

Pour lancer les tests, lancer la commande : make test  (12/15)
